Projekat se pokrece preko main metode org.example.App klase.
Sačuvane internet pakete čuvam u MySQL bazi podataka. DDL se nalazi u fajlu SQLDDL.txt.
Lozinka, username, ime baze, ime tabele su hard kodirani i nalaze se u
org.example.models.internetpackage.InternetPackage klasi. Ta klasa komunicira sa bazom podataka.

The main method of the project is in org.example.App class.
I save internet packages by putting them in an MySQL database. DDL of the database is in SQLDDL.txt file.
Username, password, database name, table name etc. are hard coded in the
org.example.models.internetpackage.InternetPackage class. That class communicates with the database.
